# deadheadhippie.net

Personal site. Built with [Hugo](https://gohugo.io/) using the [risotto](https://github.com/joeroe/risotto) theme

---

## Stack

- **Static site generator:** Hugo
- **Theme:** risotto
- **Hosting:** Cloudflare Pages

---

## Structure

```
.
├── content/        # Pages and posts (Markdown)
├── static/         # Images, GIFs, and other static assets
├── layouts/        # Custom partial overrides
├── themes/risotto/ # Theme (submodule)
└── hugo.toml       # Site configuration
```
